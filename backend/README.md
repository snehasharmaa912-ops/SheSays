# SheSays Backend

Express + MongoDB API for the SheSays platform.

## Endpoints
- GET /api/posts
- GET /api/posts/:slug
- POST /api/posts
- PUT /api/posts/:slug
- DELETE /api/posts/:slug
- POST /api/contact

## Setup
1. Copy .env.example to .env and fill in values
2. Create a free Cloudinary account at cloudinary.com and copy your Cloud Name, API Key, and API Secret into .env
3. npm install
4. npm run dev
